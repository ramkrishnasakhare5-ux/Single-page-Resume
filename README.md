# HTML Resume Website

This repository contains a **simple static resume website** created using **HTML5**.  
The project presents personal, academic, and professional details in a structured and readable format.

##  Project Overview
The resume includes the following sections:
- Header with name and professional roles
- About Me
- Technical Skills
- Work Experience
- Academic & Personal Projects
- Certificates and Awards
- Education Details
- Contact Information## Clone the Repository

Use the following command to clone the project:

```bash
git clone <repository-url>
```
## After cloning, move into the project folder:

```bash
cd <repository-folder-name>
```
## How to Run

Option 1: Open Directly
	•	Open index.html in any web browser.
	•	The resume will load immediately.

Option 2: Using VSCode and Liveserver
	1.	Open file in VS code.
	2.	Click on 'Go Live' on bottom right corner.
	3.	Go to:
  ```bash
http://localhost:3000
```
You can access the resume from there.
